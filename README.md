# Appearance Switcher

## Overview
MacBook Air / Pro のディスプレイに覗き見防止フィルムを貼ると、外観モードがライトよりダークの方が見やすいが、外部ディスプレイに繋いでいるときはライトの方が見やすいってこと、ありませんか？
私はそうでした。なので、外観モードの切り替えを一日に何度も手動で行なっていましたが、何度も行うと手間だなあとなったので、ワンクリックで切り替えられるアプリを作りました。

## Features
- macOSの外観モード（ライト / ダーク）をワンクリックで切り替え。
- 外部ディスプレイ接続時に外観モードがダークだったら、ライトに切り替えるように案内。
- 外部ディスプレイ接続時に外観モードがライトだったら、ダークに切り替えるように案内。
