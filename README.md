# appearance-switcher

macOSの外観モード(ダークモード/ライトモード)を切り替えるAppleScriptです。

## 概要

このスクリプトは、macOSの外観設定を自動的にトグル(切り替え)します。現在がダークモードの場合はライトモードに、ライトモードの場合はダークモードに切り替わります。

## 機能

- 現在の外観モードを検出
- ダークモード ⇄ ライトモードの切り替え
- ワンクリックでの実行

## 使い方

### 1. スクリプトエディタから実行

1. `AppearanceSwitcher.scpt` をダブルクリックしてスクリプトエディタで開く
2. 実行ボタン(▶️)をクリック

### 2. アプリケーションとして書き出し

1. スクリプトエディタで開く
2. 「ファイル」→「書き出す」を選択
3. ファイルフォーマットを「アプリケーション」に設定
4. 保存した.appファイルをダブルクリックで実行

### 3. ショートカットキーで実行

1. 「システム設定」→「キーボード」→「キーボードショートカット」を開く
2. 「サービス」→「一般」でスクリプトを登録
3. または、Automatorを使用してサービスとして登録し、ショートカットを割り当てる

## 動作環境

- macOS 10.14 (Mojave) 以降
  - ダークモード機能が導入されたバージョン

## 技術詳細

スクリプトは以下の処理を行います:

1. `System Events` のアクセシビリティ権限を使用
2. `appearance preferences` から現在のダークモード設定を取得
3. 現在の状態を反転させて設定

## 権限

初回実行時に、「システムイベント」へのアクセス権限を求められる場合があります。「システム設定」→「プライバシーとセキュリティ」→「アクセシビリティ」で権限を許可してください。
